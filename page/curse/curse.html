Music Box

HideShow

- 効果 -

  • 20m圏内にいるゴーストが階層を問わず歌う。
  • オルゴールの3m圏内では、1秒毎に正気度が5%ずつ減る。
  • ゴーストがオルゴールの5m圏内に入るとオルゴールに実体化して向かって来て、5秒後に呪いのハントが発生。
  • オルゴールの3m圏内に正気度が0%の調査員がいた場合、すぐに音楽が中断し、呪いのハントが発生。

- 注意点 -

  • 床などに設置する際、Fキーで設置せずに落とすボタンで放り投げてしまうと、即座に呪いのハントが発生。
  • 最後まで曲を聴いたときの正気度現象量は80%。
  • オルゴールは遠方にいるゴーストを引き寄せるわけではなく、あくまで実体化したゴーストを引き寄せるだけ。

- 活用的な使用方法 -

  • 呪いのハントを起こしたいなら、ある程度離れた場所で使い始めゴーストルームの近くに置くのが比較的安全。
  • 実体化ゴーストが設置されたオルゴールに接触しても、近くに正気度が0%の調査員がいない限り、5秒間の実体化イベントが終わるまでハントに移行しない。これを利用してゴーストの写真も簡単に撮れる。

Ouija Board

HideShow

- 効果 -

  • 文字板を使ってゴーストと交信することができる。

- 注意点 -

  • 利用した後は「さようなら」と告げてプランシェットを片付ける必要がある。プランシェットをそのままにしてウィジャ盤から５ｍ遠ざかると、ウィジャ盤が赤く燃えて壊れ呪いのハントが発生します。

- 活用的な使用方法 -

  • ハントを起こしたい場合、「かくれんぼ」というか、プランシェットを出したままウィジャ盤から離れれば正気度を減らさずに呪いのハントを発生させることが可能。
  • ウィジャ盤が質問を認識した場合はEMFレベル2の反応が出る。EMFレベル5を証拠に持つゴーストの場合、正気度減少が少ない質問を繰り返すことで、比較的楽にEMFレベル5の反応を取ることが可能。

- 正気度系の質問について -

  • 正気度系の質問のコストはすべて 5%。
  • 返答は、質問による正気度が減った後の正気度。

Voodoo Doll

HideShow

- 効果 -

  • 使用することでゴーストの干渉を促すことができる。
  • 使用すると10本のピンの内1本が押し込まれる。

- 注意点 -

  • 心臓のピンが押し込まれた場合、強制ハントが発生。
  • 心臓は10%、その他は5%正気度が減少する。
  • プレイヤーの正気度が10%未満の場合、ピンが全て押し込まれ、呪いのハントが発生する。

- 活用的な使用方法 -

  • どこかに隠れながら使用し、心臓に刺さったピンが押し込まれた後でゴーストルームに持っていき、干渉を誘発する、という使い方もできる。

Haunted Mirror

HideShow

- 効果 -

  • 使用すると、鏡の中にゴーストルームの風景が映る。

- 注意点 -

  • 最短で連打しても最低20%の正気度が失われる。
  • 鏡を覗いている間に正気度が0%になるor正気度が20%未満の状態で鏡を覗こうとするとその時点で鏡が割れ、ゴーストの現在地から呪いのハントが発生する。
  • プレイヤーの正気度が10%未満の場合、ピンが全て押し込まれ、呪いのハントが発生する。

Summoning Circle

HideShow

- 効果 -

  • 周囲に5本の蝋燭が立っており、全ての蝋燭に火を点けるとゴーストを召喚することが可能です。
  • 召喚されたゴーストは5秒間その場で拘束され、拘束が終了すると呪いのハントが開始する。

- 注意点 -

  • 蝋燭に火を点けると魔法陣の3m圏内にいる全てのプレイヤーの正気度が16%減少。全てつけると正味80%の減少
  • 蝋燭を点けている最中に、3m圏内にいる調査員が正気度が0%になってしまった場合はその時点で火が消えてしまい、使用することができなくなる。
  • 5本目の蝋燭を点ける際に正気度が16%以下である場合は使用することができますが、この場合はゴーストが召喚された直後に拘束時間なしでハントが開始される。

Monkey Paw

HideShow

- 効果 -

  • 手にした状態で願い事を口にすることで、代償と引き換えに願い事を叶えてくれる。
  • 願い事ができる回数は難易度によって決まり、アマチュアであれば5回、セミプロであれば4回、プロ以上であれば3回。(指の本数でもわかる)

- 注意点 -

  • 1度の調査で同じ願い事を複数回利用することはできない

- 質問一覧 -

== ゴーストが見たい ==

  • 正:ゴーストがゴーストの現在位置で実体化
  • 負:視界が暗くなり、5秒後に呪いのハントが開始。

== 怪奇現象が見たい ==

  • 正:ゴーストの干渉、徘徊、特殊能力の確率が2分間2倍。
  • 負:ブレーカーが故障し、玄関が2分間開かなくなる。

== 閉じ込めたい ==

  • 正:ゴーストをゴーストルームに1分間閉じ込める
  • 負:プレイヤーのいる部屋のドアも1分間開かなくなる。その後は通常のハントが開始する。

== 正気を戻りたい ==

  • 正:全てのプレイヤーの正気度を50%にする。
  • 負:ゴーストルームが変更。暗闇での正気度の減少量が50%増える。

== 身を守りたい ==

  • 正:近くにある塞がれた隠れ場所が開放される。
  • 負:部屋の電球が割れ、ハント時に電子機器や声が届く距離に制限がなくなる。

== 帰りたい ==

  • 正:ハント中に使うと玄関を開けられるようになる。
  • 負:5秒間移動速度が減少、その間視界が狭まる。

== 生き返りたい ==

  • 正:死亡したプレイヤーを生き返らせる。
  • 負:50%の確率で使用者が死亡する。

== 知識が欲しい ==

  • 正:ジャーナルで証拠がひとつ除外される。
  • 負:プレイヤーの近くで呪いのハントが開始。視界が暗くなり音もほとんど聞こえなくなる（死ぬまで永続）。

== (晴れ/雨/霧/夜明け/雪)になって ==

  • 正:天候を即座に変更する。
  • 負:正気度を25%消費。（足りないときは使えない）。

== ゴーストが見たい ==

  • 正:ゴーストがゴーストの現在位置で実体化
  • 負:視界が暗くなり、5秒後に呪いのハントが開始。

== ゴーストが見たい ==

  • 正:ゴーストがゴーストの現在位置で実体化
  • 負:視界が暗くなり、5秒後に呪いのハントが開始。
